#
A simple command-line app to track daily expenses using Python.

## Features
- Add expenses with category & notes
- View all expenses with total spent
- Auto-saves data to CSV file
- Date tracking for each entry

## Tech Stack
- Python 3
- CSV module for data storage

## How to Run
1. Clone this repo: `git clone https://github.com/hariprasad-cse/expense-tracker-cli`
2. Run: `python main.py`
3. Choose options to add/view expenses
